# End-FullStack-End

This Repo serves as battle tested , opinionated reference for Full Stack App Scaffolding and development. Ever Since AI eveolved from pair programming to full end to end product development if the specifications and prompts are clearly mentioned, Developers and even PMs tend to build projects without writing single piece of code. Students , Interns and Freshers are also using AI to learn and implement projects with AI , bypassing the conventional way of reading and implementing fmro docs. However when we want to improvise the POC or scale up to production grade , that's when the bottle neck arises. Improper prompt leading to unorganized project structure which acts as a barrier to scaling up the app and also poor choice of packages making the application non functioning. In order to overcome this I have created this repo for AI reference and also a quick reference for beginners and experienced developers to decide upon the package choices and arrive to a solid working conclusion.

I have planned to cover the most popular tech stacks used in web development namely , Node , Python and Java. Anyone is open to fork and extend it to any other language or frameworks.
